# Rock Paper Scissors Game 🎮

A simple console-based Rock, Paper, Scissors game in C++.

# 📋 How to Play :

- You play against the computer.
- Choose one of the options:
  - [1] Stone 🪨
  - [2] Paper 📄
  - [3] Scissors ✂
- The computer makes a random choice.
- The game will show who won each round and the final winner.

# 🔁 Features : 

- Multiple rounds support.
- Score tracking for player, computer, and draws.
- Random computer choice using rand().
- Color changes based on who won the round.
- Option to play again after each game.

# 💻 Language :

C++
